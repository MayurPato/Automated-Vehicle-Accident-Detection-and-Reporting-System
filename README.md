# Automated-Vehicle-Accident-Detection-and-Reporting-System

Currently, this project is in the development phase. At this point, I've created a module that locates police stations within the quickest reach in the event of an accident being detected, using the [Google Maps API](https://github.com/googlemaps/google-maps-services-python). The next step (not yet developed) involves automating the process of alerting the closest police station, possibly through a phone call, about the accident. 

I am also working on accurately identifying accidents from CCTV footage, eliminating the need for sensors.
